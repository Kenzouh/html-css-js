String Methods

  • .charAt() = get character at the specified index.
  • .indexOf() = returns the index of the value.
  • .lastIndexOf() = returns the LAST INDEX where the value can be last found or its last occurrence.
  • .trim() = removes blank space at the start.
  • .toUpperCase() = transforms the string to UPPERCASE.
  • .toLowerCase() = transforms the string to lowercase.
  • .startsWith() = checks if the string STARTS WITH the value stated.
  • .includes() = checks if the string INCLUDES the value stated.
  • .endsWith() = checks if the string ENDS WITH the value stated.
  • .length = Not a method. Returns the length of a string.
