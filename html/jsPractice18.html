Equality Symbols

  • = → Assignment Operator
  • == → Comparison Operator (compares the 2 VALUES if they're EQUAL)
  • === → Strict equality Operator (Compares the 2 VALUES if they're EQUAL and have the SAME DATA TYPE)
  • != → Inequality Operator
  • !== → Strict Inequality Operator
